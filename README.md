# CCLIB

## Installation

`git clone https://github.com/Flootersmm/cclib`

`cd` into repo directory

`make all`

`make install`

## Testing

`make test`

## Usage

`gcc -o my_project my_project.c -lcclib`

## Documentation

Located at docs/html/index.html

Regenerate with `make docs`.

## Cleaning

`make clean`
